1. What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML elements. It controls the layout, colors, fonts, spacing, and visual appearance of web pages.

2. CSS Syntax

CSS rules consist of a selector and a declaration block:

selector { property: value; }

  • selector ‚Üí HTML element you want to style.
  • property ‚Üí the style feature (color, font-size, etc.)
  • value ‚Üí the specific setting of that property.

3. Types of CSS

  • Inline CSS: directly inside the HTML element using the style attribute.
  • Internal CSS: inside a <style> tag in the HTML <head>.
  • External CSS: in a separate .css file linked with <link>.

4. Common CSS Selectors

  • element ‚Üí targets all elements of that type.
  • .class ‚Üí targets elements with the class.
  • #id ‚Üí targets the element with the specific id.
  • * ‚Üí universal selector, targets all elements.

5. CSS Box Model

Every HTML element is a rectangular box made of: content, padding, border, and margin. Understanding the box model is crucial for layout and spacing.

6. CSS Layouts

CSS offers multiple layout techniques: Flexbox, Grid, and positioning to organize elements on the page efficiently.
