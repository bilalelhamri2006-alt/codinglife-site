Chapter 1: CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and positioning of HTML elements.

CSS Syntax

CSS consists of selectors and declaration blocks:

CSS Syntax Structure

selector {
  property: value;
  property: value;
}

Ways to Add CSS

There are three ways to include CSS in your HTML:

1. Inline CSS

<h1 style="color: blue; font-size: 24px;">Title</h1>

2. Internal CSS

<style>
  h1 {
    color: blue;
    font-size: 24px;
  }
</style>

3. External CSS

<link rel="stylesheet" href="styles.css">

Chapter 2: Colors and Backgrounds

Color Properties

CSS provides multiple ways to specify colors:

Color Examples

.element {
  color: red;                    /* Named color */
  color: #ff0000;                /* Hex color */
  color: rgb(255, 0, 0);         /* RGB */
  color: rgba(255, 0, 0, 0.5);   /* RGB with opacity */
  color: hsl(0, 100%, 50%);       /* HSL */
}

Background Properties

Control element backgrounds with various properties:

Background Examples

.box {
  background-color: #f0f0f0;
  background-image: url('image.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}

Chapter 3: Text and Fonts

Text Styling

Control text appearance with CSS properties:

Text Properties

.text {
  font-family: 'Arial', sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
  text-align: center;
  text-decoration: underline;
  line-height: 1.6;
  letter-spacing: 1px;
}

Google Fonts

Use web fonts from Google Fonts:

Using Google Fonts

<!-- In HTML head -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

/* In CSS */
body {
  font-family: 'Roboto', sans-serif;
}

Chapter 4: Box Model

Understanding the Box Model

Every element in CSS is a box with content, padding, border, and margin.

Box Model Properties

.box {
  /* Content */
  width: 200px;
  height: 100px;
  
  /* Padding */
  padding: 20px;
  
  /* Border */
  border: 2px solid #000;
  
  /* Margin */
  margin: 15px;
}

Box Sizing

Control how element dimensions are calculated:

Box Sizing Examples

.content-box {
  box-sizing: content-box; /* Default */
  width: 200px;
  padding: 20px; /* Total width = 240px */
}

.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px; /* Total width = 200px */
}

Chapter 5: Flexbox Layout

Flexbox Basics

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

Flex Container Properties

.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
  gap: 10px;
}

Flex Items

Control individual flex items:

Flex Item Properties

.item {
  flex: 1;
  align-self: flex-start;
  order: 2;
}

Chapter 6: Responsive Design

Media Queries

Media queries allow you to apply different styles for different devices:

Media Query Examples

/* Mobile First */
.container {
  padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 20px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    padding: 40px;
  }
}

Responsive Units

Use relative units for responsive layouts:

CSS Units

.responsive {
  width: 100%;        /* Percentage */
  font-size: 1rem;     /* Root element */
  padding: 1em;       /* Parent element */
  width: 50vw;        /* Viewport width */
  height: 50vh;       /* Viewport height */
}
