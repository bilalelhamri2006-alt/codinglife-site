Chapter 1: Python Basics

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data science, AI, and automation.

Key Features

  • Easy to Learn: Simple and readable syntax
  • Versatile: Web, data science, AI, automation
  • Large Community: Extensive libraries and frameworks
  • Cross-platform: Runs on Windows, Linux, macOS
  • Open Source: Free to use and modify

Installing Python

Download Python from the official website or use package managers:

Installation Methods

# Windows: Download from python.org
# macOS: Using Homebrew
brew install python

# Linux (Ubuntu/Debian)
sudo apt update
sudo apt install python3 python3-pip

# Verify installation
python3 --version
pip3 --version

Basic Python Syntax

This is a multi-line

Chapter 2: Variables and Data Types

Variables in Python

Python variables are dynamically typed - no need to declare type:

Variable Declaration

# Variable assignment
name = "John Doe"
age = 25
height = 5.9
is_student = True

# Output variables
print("Name:", name)
print("Age:", age)
print(f"My name is {name} and I am {age} years old")

Data Types

Python has several built-in data types:

Python Data Types

# String
text = "Hello Python"

# Integer
number = 42

# Float
pi = 3.14159

# Boolean
is_valid = True

# List
fruits = ["apple", "banana", "orange"]

# Tuple (immutable)
coordinates = (10.5, 20.3)

# Dictionary
person = {"name": "Alice", "age": 30}

# Check data type
print(type(text))      # <class 'str'>
print(type(number))    # <class 'int'>

Chapter 3: Control Structures

Conditional Statements

Control program flow with if-elif-else statements:

If-Elif-Else

age = 20

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

# Multiple conditions
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"Grade: {grade}")

Loops

Repeat actions with for and while loops:

Loop Examples

# For loop with range
for i in range(5):
    print(f"Number: {i}")

# For loop with list
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# While loop
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# Loop control statements
for i in range(10):
    if i == 5:
        break      # Exit loop
    if i % 2 == 0:
        continue   # Skip iteration
    print(i)

Chapter 4: Functions and Data Structures

Functions

Reusable blocks of code with def keyword:

Function Examples

# Simple function
def greet(name):
    return f"Hello, {name}!"

# Function with default parameter
def calculate_area(radius, pi=3.14159):
    return pi * radius ** 2

# Function with multiple return values
def get_user_info():
    name = "John"
    age = 25
    return name, age

# Calling functions
print(greet("Alice"))              # Hello, Alice!
print(calculate_area(5))          # 78.53975
username, user_age = get_user_info()

Data Structures

Advanced data structures in Python:

Lists, Tuples, Dictionaries

# List (mutable)
fruits = ["apple", "banana", "orange"]
fruits.append("grape")        # Add item
fruits.remove("banana")      # Remove item
print(fruits[0])             # Access first item

# Tuple (immutable)
colors = ("red", "green", "blue")
print(colors[1])             # Access second item

# Dictionary (key-value pairs)
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
print(person["name"])         # Access value by key
person["job"] = "Engineer"   # Add new key-value

# Set (unique elements)
unique_numbers = {1, 2, 3, 3, 4}  # {1, 2, 3, 4}

Chapter 5: File Handling and Modules

File Operations

Reading and writing files in Python:

File Handling

# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python!\n")
    file.write("This is a text file.")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
with open("example.txt", "r") as file:
    for line in file:
        print(line.strip())  # strip() removes newline characters

# Appending to a file
with open("example.txt", "a") as file:
    file.write("\nAppended text!")

Modules and Packages

Using built-in and external modules:

Importing Modules

# Import entire module
import math
print(math.sqrt(16))      # 4.0

# Import specific functions
from datetime import datetime
print(datetime.now())     # Current date and time

# Import with alias
import numpy as np
import pandas as pd

# Creating your own module
# Save as mymodule.py
"""
# mymodule.py
def greet(name):
    return f"Hello, {name}!"

PI = 3.14159
"""

# Using your module
import mymodule
print(mymodule.greet("Bob"))
print(mymodule.PI)
