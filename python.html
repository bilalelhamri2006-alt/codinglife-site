1. What is Python?

Python is a popular, high-level, interpreted programming language known for its readability and versatility. It's widely used in web development, data science, AI, and automation.

  • Easy to learn with simple syntax
  • Cross-platform compatibility
  • Large standard library
  • Strong community support

2. Python Syntax

Python uses indentation to define code blocks instead of braces, making the code clean and readable.

if True:
  print("Hello Python")

This simple syntax reduces common errors and makes Python ideal for beginners.

3. Variables & Data Types

Python variables are dynamically typed, meaning you don't need to declare their type explicitly.

name = "Bilal"
age = 18
price = 19.99
is_active = True
  • String: Text data like "Hello"
  • Integer: Whole numbers like 10, -5, 0
  • Float: Decimal numbers like 3.14, 2.5
  • Boolean: True or False values
  • List: Ordered collection like ["Apple", "Banana"]

4. Operators

Python supports various operators for performing operations on variables and values.

  • Arithmetic: + - * / % ** //
  • Comparison: == != < > <= >=
  • Logical: and, or, not
  • Assignment: = += -= *= /=
x = 10
y = 5
print(x + y) # Output: 15

5. Conditional Statements

Conditional statements allow your program to make decisions based on certain conditions.

age = 20
if age > 18:
  print("Adult")
else:
  print("Minor")

You can also use elif for multiple conditions:

if age < 13:
  print("Child")
elif age < 20:
  print("Teenager")
else:
  print("Adult")

6. Loops

Loops allow you to execute a block of code repeatedly.

for i in range(1,6):
  print(i)

This will print numbers from 1 to 5. Python also supports while loops:

i = 1
while i <= 5:
  print(i)
  i += 1

7. Functions

Functions are reusable blocks of code that perform a specific task.

def greet(name):
  return "Hello " + name

print(greet("Bilal")) # Output: Hello Bilal

Functions can take parameters and return values, making your code modular and reusable.

8. Lists & Loops

Lists are ordered collections that can hold items of different data types.

fruits = ["Apple", "Banana", "Orange"]
for f in fruits:
  print(f)

This will print each fruit on a separate line. Lists are mutable, meaning you can change their content.

9. Dictionaries

Dictionaries store data as key-value pairs, making it easy to retrieve values by their keys.

person = {"name": "Bilal", "age": 18, "city": "Rabat"}
print(person["name"]) # Output: Bilal

You can add, modify, or delete dictionary items:

person["country"] = "Morocco" # Add new key-value pair
person["age"] = 19 # Modify existing value
del person["city"] # Delete a key-value pair

10. Modules & Libraries

Python has a rich standard library and supports many third-party libraries for various tasks.

import math
print(math.sqrt(16)) # Output: 4.0

Popular Python libraries include:

  • NumPy: For numerical computations
  • Pandas: For data analysis
  • Django/Flask: For web development
  • Matplotlib: For data visualization
