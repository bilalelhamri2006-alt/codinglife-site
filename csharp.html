1. What is C#?

C# is a modern, object-oriented programming language developed by Microsoft for building Windows, web, and mobile applications.

2. Setting up C# Environment

  • Install Visual Studio
  • Install .NET SDK
  • Create a new C# project

3. Hello World

Write your first C# program:

Console.WriteLine("Hello, World!");

4. Variables

  • int, double, string, bool
  • Declaration: int age = 25;

5. Data Types

  • int, float, double, decimal, char, string, bool

6. Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !

7. Conditional Statements

  • if, else if, else
  • switch statements

8. Loops

  • for, foreach
  • while, do-while

9. Arrays

  • Declaration: int[] numbers = {1,2,3};
  • Access: numbers[0]

10. Methods

Reusable code blocks:

void SayHello() { Console.WriteLine("Hello"); }

11. Classes and Objects

Object-oriented programming concept.

class Person { public string Name; }

12. Inheritance

Derive a class from another class:

class Student : Person { }

13. Polymorphism

Methods can have different implementations.

14. Encapsulation

Protect data using properties and access modifiers.

15. Interfaces

Define contracts for classes:

interface IAnimal { void Speak(); }

16. Exception Handling

Manage runtime errors:

try { } catch(Exception e) { }

17. Generics

Create type-safe classes and methods:

List<int> numbers = new List<int>();

18. Delegates and Events

Use delegates for callbacks and events for notifications.

19. LINQ

Query collections easily using LINQ syntax.

20. Best Practices

  • Use meaningful names
  • Keep methods small
  • Comment your code
  • Follow SOLID principles
