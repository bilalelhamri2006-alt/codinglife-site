Chapter 1: PHP Basics

What is PHP?

PHP (Hypertext Preprocessor) is a server-side scripting language designed for web development. It's embedded in HTML and executed on the server.

Key Features

  • Server-side: Executed on the server before sending to browser
  • Open Source: Free to use and modify
  • Cross-platform: Runs on Windows, Linux, macOS
  • Database Support: Native support for MySQL, PostgreSQL, etc.
  • Large Community: Extensive documentation and support

Chapter 2: Variables and Data Types

Variables in PHP

PHP variables start with $ followed by the variable name:

Variable Declaration

<?php
// Variable declaration
$name = "John Doe";
$age = 25;
$height = 5.9;
$isStudent = true;

// Output variables
echo "Name: " . $name;
echo "Age: " . $age;

// Variable interpolation in double quotes
echo "My name is $name and I am $age years old";
?>

Data Types

PHP supports several data types:

PHP Data Types

<?php
// String
$string = "Hello World";

// Integer
$integer = 42;

// Float (double)
$float = 3.14;

// Boolean
$bool = true;

// Array
$array = array("apple", "banana", "orange");

// Null
$null = null;

// Check data type
echo gettype($string); // string
echo var_dump($integer); // int(42)
?>

Chapter 3: Control Structures

Conditional Statements

Control program flow based on conditions:

If-Else Statements

<?php
$age = 20;

if ($age >= 18) {
    echo "You are an adult";
} elseif ($age >= 13) {
    echo "You are a teenager";
} else {
    echo "You are a child";
}

// Switch statement
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Start of work week";
        break;
    case "Friday":
        echo "Weekend is coming!";
        break;
    default:
        echo "Regular day";
}
?>

Loops

Repeat actions multiple times:

Loop Examples

<?php
// For loop
for ($i = 0; $i < 5; $i++) {
    echo "Number: " . $i . "<br>";
}

// While loop
$count = 0;
while ($count < 3) {
    echo "Count: " . $count . "<br>";
    $count++;
}

// Foreach loop (for arrays)
$fruits = ["apple", "banana", "orange"];
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}
?>

Chapter 4: Functions and Arrays

Functions

Reusable blocks of code:

Function Examples

<?php
// Simple function
function greet($name) {
    return "Hello, " . $name . "!";
}

// Function with default parameter
function calculateArea($radius, $pi = 3.14) {
    return $pi * $radius * $radius;
}

// Calling functions
echo greet("Alice"); // Hello, Alice!
echo calculateArea(5); // 78.5

// Built-in functions
echo strtoupper("hello"); // HELLO
echo strlen("PHP"); // 3
?>

Arrays

Collections of data:

Array Examples

<?php
// Indexed array
$colors = ["red", "green", "blue"];
$colors = array("red", "green", "blue");

// Associative array
$person = [
    "name" => "John",
    "age" => 30,
    "city" => "New York"
];

// Accessing array elements
echo $colors[0]; // red
echo $person["name"]; // John

// Array functions
array_push($colors, "yellow"); // Add element
count($colors); // Get length
sort($colors); // Sort array
?>

MySQL Database Connection

Connecting to MySQL database with PHP:

MySQLi Connection

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "my_database";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

// Create table
$sql = "CREATE TABLE users (
    id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(30) NOT NULL,
    email VARCHAR(50),
    reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)";

if ($conn->query($sql) === TRUE) {
    echo "Table created successfully";
}

// Insert data
$sql = "INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com')";
$conn->query($sql);

// Close connection
$conn->close();
?>
