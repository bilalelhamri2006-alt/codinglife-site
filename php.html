1. What is PHP?

PHP (Hypertext Preprocessor) is a popular open-source server-side scripting language used to create dynamic websites, handle forms, interact with databases, and manage sessions.

  • Runs on server before sending HTML to browser.
  • Can be embedded into HTML code.
  • Works with databases like MySQL, PostgreSQL, etc.

2. PHP Syntax

PHP code is placed between <?php ... ?> tags. Each statement ends with a semicolon.

<?php echo "Hello World!"; ?>

  • Statements end with ;
  • PHP is case-insensitive for keywords (echo, print)
  • Comments: // single line, /* multi-line */

3. Variables & Data Types

Variables start with $ and are dynamically typed.

$name = "Bilal"; $age = 18; $price = 19.99; $isActive = true;

  • String: Text enclosed in quotes.
  • Integer: Whole numbers.
  • Float: Decimal numbers.
  • Boolean: true or false.
  • Array: Collection of values $fruits = ["Apple","Banana"];

4. Operators

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !

$x = 10; $y = 5; echo $x + $y; // 15

5. Conditional Statements

Control the flow using if, else, elseif, switch:

if($age > 18){ echo "Adult"; } else { echo "Minor"; }

6. Loops

Repeating actions using for, while, do-while, foreach:

for($i=1;$i<=5;$i++){ echo $i; }

7. Functions

Functions allow code reusability:

function greet($name){
  return "Hello $name!";
}
echo greet("Bilal");

8. Forms & Superglobals

PHP handles form data using $_POST or $_GET:

<form method="post">
  <input name="username">
  <button type="submit">Send</button>
</form>
$name = $_POST['username'];

9. Arrays & Loops

Arrays can be looped using foreach:

$fruits = ["Apple","Banana","Orange"];
foreach($fruits as $fruit){ echo $fruit; }

10. PHP & MySQL

PHP can connect to databases to store, retrieve, and manipulate data:

$conn = new mysqli($host,$user,$pass,$db);
$result = $conn->query("SELECT * FROM users");

11. Sessions & Cookies

PHP can manage user sessions:

  • session_start(); $_SESSION['user']="Bilal";
  • setcookie("name","Bilal",time()+3600);
