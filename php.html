1. What is PHP?

PHP is a popular server-side language used to build dynamic websites.

  • Runs on server
  • Embeds inside HTML
  • Connects to databases

2. PHP Syntax

PHP code is written inside <?php ?> tags.

<?php echo "Hello"; ?>

3. Variables & Data Types

Variables start with $ and are dynamically typed.

$name = "Bilal"; $age = 18;

4. Operators

$x = 10; $y = 5; echo $x + $y;

5. Conditional Statements

if($age > 18){ echo "Adult"; }

6. Loops

for($i=1;$i<=5;$i++){ echo $i; }

7. Functions

function greet($name){ return "Hello $name!"; }

8. Forms & Superglobals

<form method="post"><input name="username"></form>

9. Arrays

$fruits = ["Apple","Banana"];

10. PHP & MySQL

$conn = new mysqli($h,$u,$p,$d);

11. Sessions & Cookies

session_start(); $_SESSION['user']="Bilal";
