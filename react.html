1. What is React?

React is a JavaScript library for building user interfaces, developed by Facebook. It allows developers to create reusable UI components that make web applications faster and more organized.

2. Components in React

Components are the heart of React. Each component is an independent, reusable piece of UI.

4. Virtual DOM

React uses a virtual DOM to update only the changed parts of the page, improving performance significantly.

5. Hooks in React

Hooks (like useState and useEffect) allow you to use state and other React features without writing a class.
