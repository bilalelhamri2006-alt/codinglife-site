1. What is SQL?

SQL (Structured Query Language) is used to communicate with databases. It allows you to create, read, update, and delete data (CRUD operations).

2. Basic SQL Syntax

SQL statements have a standard structure:

SELECT column1, column2 FROM table_name WHERE condition;

3. Creating Tables

Use CREATE TABLE to define a new table:

CREATE TABLE Users (
  ID INT PRIMARY KEY,
  Name VARCHAR(50),
  Email VARCHAR(100)
);

4. Inserting Data

Use INSERT INTO to add records:

INSERT INTO Users (ID, Name, Email)
VALUES (1, 'Bilal', 'bilal@example.com');

5. Retrieving Data

Use SELECT to read data:

SELECT Name, Email FROM Users WHERE ID = 1;

  • SELECT * : select all columns
  • WHERE : filter records
  • ORDER BY : sort results

6. Updating Data

Use UPDATE to modify existing records:

UPDATE Users SET Email='newemail@example.com' WHERE ID=1;

7. Deleting Data

Use DELETE to remove records:

DELETE FROM Users WHERE ID=1;

8. Joins

Joins combine rows from multiple tables:

SELECT Orders.ID, Users.Name FROM Orders
INNER JOIN Users ON Orders.UserID = Users.ID;

9. Aggregate Functions

  • COUNT() - count rows
  • SUM() - sum values
  • AVG() - average value
  • MAX() - maximum value
  • MIN() - minimum value

SELECT COUNT(*) FROM Users;

10. Constraints

Ensure data integrity using constraints:

  • PRIMARY KEY - unique identifier
  • FOREIGN KEY - link between tables
  • NOT NULL - no empty values
  • UNIQUE - unique values

CREATE TABLE Orders (
  ID INT PRIMARY KEY,
  UserID INT,
  FOREIGN KEY(UserID) REFERENCES Users(ID)
);
