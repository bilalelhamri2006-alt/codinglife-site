1. What is JavaScript?

JavaScript is a programming language used to make web pages interactive. It can manipulate HTML and CSS, handle events, and communicate with servers.

2. JavaScript Syntax

JavaScript uses statements, expressions, and functions to perform actions. Example:

let x = 10;

3. Variables and Data Types

Variables store data and can have types like string, number, boolean, object, or array.

Example:

let name = "Bilal"; let age = 20;

4. Functions

Functions are reusable blocks of code:

function greet(){ console.log("Hello!"); }

Call the function: greet();

5. Events

JavaScript can respond to user actions using events, like click, mouseover, keydown:

button.addEventListener("click", function(){ alert("Clicked!"); });

6. DOM Manipulation

The DOM (Document Object Model) represents the web page. JavaScript can change content, styles, or structure dynamically:

document.getElementById("demo").innerHTML = "Hello JavaScript";
