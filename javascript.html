Chapter 1: JavaScript Basics

What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It's an essential part of web development alongside HTML and CSS.

Variables and Data Types

JavaScript has three ways to declare variables:

Variables Declaration

let name = "John";      // Can be reassigned
const age = 25;         // Cannot be reassigned
var city = "New York";  // Old way (avoid)

// Data Types
let string = "Hello";
let number = 42;
let boolean = true;
let array = [1, 2, 3];
let object = {name: "John"};
let nullValue = null;
let undefinedValue;

Chapter 2: Operators and Functions

Operators

JavaScript has various operators for operations:

Common Operators

// Arithmetic
let sum = 10 + 5;      // 15
let product = 10 * 5;  // 50

// Comparison
let isEqual = 10 === 10;   // true
let isGreater = 10 > 5;   // true

// Logical
let and = true && false;  // false
let or = true || false;   // true

Functions

Functions are reusable blocks of code:

Function Declarations

// Function Declaration
function greet(name) {
  return "Hello, " + name;
}

// Function Expression
const multiply = function(a, b) {
  return a * b;
};

// Arrow Function (ES6)
const divide = (a, b) => a / b;

Chapter 3: Control Structures

Conditional Statements

Control program flow based on conditions:

If-Else Statements

let age = 18;

if (age >= 18) {
  console.log("Adult");
} else if (age >= 13) {
  console.log("Teenager");
} else {
  console.log("Child");
}

// Switch Statement
let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of week");
    break;
  case "Friday":
    console.log("Weekend is near");
    break;
  default:
    console.log("Regular day");
}

Loops

Repeat actions multiple times:

Loop Examples

// For Loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While Loop
let count = 0;
while (count < 3) {
  console.log(count);
  count++;
}

// For...of Loop (Arrays)
const fruits = ["apple", "banana", "orange"];
for (const fruit of fruits) {
  console.log(fruit);
}

Chapter 4: Arrays and Objects

Arrays

Arrays store multiple values in a single variable:

Array Methods

const colors = ["red", "green", "blue"];

// Adding/Removing
colors.push("yellow");      // Add to end
colors.pop();              // Remove from end
colors.unshift("purple");   // Add to start

// Useful Methods
const doubled = colors.map(color => color + "!");
const filtered = colors.filter(color => color.length > 3);
const found = colors.find(color => color === "red");

Objects

Objects store key-value pairs:

Object Examples

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  hobbies: ["reading", "coding"],
  
  // Method
  getFullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

// Accessing properties
console.log(person.firstName);        // Dot notation
console.log(person["lastName"]);     // Bracket notation
console.log(person.getFullName());   // Method call

Chapter 5: DOM Manipulation

What is DOM?

The Document Object Model (DOM) represents the page structure and allows JavaScript to interact with HTML elements.

DOM Selection Methods

// Selecting elements
const element = document.getElementById("myId");
const elements = document.querySelectorAll(".myClass");
const firstElement = document.querySelector("div");

// Modifying content
element.textContent = "New text";
element.innerHTML = "<strong>Bold text</strong>";

// Changing styles
element.style.color = "red";
element.style.backgroundColor = "yellow";

Event Handling

Respond to user interactions:

Event Listeners

// Adding event listeners
element.addEventListener("click", function() {
  console.log("Element clicked!");
});

// Event object
element.addEventListener("click", function(event) {
  console.log("Clicked at: " + event.clientX + ", " + event.clientY);
});

// Form events
form.addEventListener("submit", function(event) {
  event.preventDefault(); // Prevent form submission
  // Handle form data
});
