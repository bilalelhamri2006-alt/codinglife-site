1. What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 engine. It allows developers to run JavaScript on the server side to build scalable network applications.

2. Node.js Modules

Node.js uses modules to organize code. Modules can be built-in, custom, or installed via npm (Node Package Manager).

  • Built-in modules: fs, http, path
  • Custom modules: require('./myModule')
  • Third-party modules: npm install express

3. npm - Node Package Manager

npm is the default package manager for Node.js. It helps you install, update, and manage dependencies for your Node.js projects.

4. Creating a Server

Node.js allows you to create a web server easily using the http module.

{`const http = require('http'); const server = http.createServer((req,res)=>{ res.end('Hello Node!'); }); server.listen(3000);`}

5. Express.js

Express.js is a popular framework for Node.js that simplifies server creation and routing.

{`const express = require('express'); const app = express(); app.get('/', (req,res)=> res.send('Hello Express')); app.listen(3000);`}

6. Asynchronous Programming

Node.js uses asynchronous, non-blocking I/O to handle multiple requests efficiently using callbacks, promises, or async/await.
