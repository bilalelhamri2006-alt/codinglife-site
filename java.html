1. What is Java?

Java is a high-level, object-oriented, platform-independent programming language. It is widely used for web development, mobile apps, desktop applications, and enterprise systems.

2. Java Syntax

Java code structure typically has classes and methods:

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello Java");
  }
}

3. Variables & Data Types

  • int: Whole numbers
  • double: Decimal numbers
  • String: Text
  • boolean: true or false

int age = 18;
double price = 19.99;
String name = "Bilal";
boolean isActive = true;

4. Operators

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !

int x = 10, y = 5;
System.out.println(x + y); // 15

5. Conditional Statements

Use if, else if, else to control the flow:

int age = 20;
if(age > 18) {
  System.out.println("Adult");
} else {
  System.out.println("Minor");
}

6. Loops

Use for, while, and do-while:

for(int i=1; i<=5; i++) {
  System.out.println(i);
}

7. Methods

Methods are reusable blocks of code:

public static int sum(int a, int b) {
  return a + b;
}
System.out.println(sum(5, 10));

8. Arrays

Arrays store multiple values:

int[] numbers = {1,2,3,4,5};
System.out.println(numbers[0]); // 1

9. Object-Oriented Programming

Java supports OOP concepts: Class, Object, Inheritance, Polymorphism, Encapsulation.

class Person {
  String name;
  int age;
  Person(String name, int age) {
    this.name = name;
    this.age = age;
  }
}

10. Packages & Libraries

Use built-in or external packages:

import java.util.Scanner;
Scanner sc = new Scanner(System.in);
