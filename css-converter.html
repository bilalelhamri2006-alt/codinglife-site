CSS Units Reference

Relative Units

  • em Relative to parent font-size
  • rem Relative to root font-size
  • vw 1% of viewport width
  • vh 1% of viewport height
  • % Percentage of parent element

Absolute Units

  • px Pixels (1px = 1/96th of 1in)
  • cm Centimeters
  • mm Millimeters
  • in Inches (1in = 96px = 2.54cm)
  • pt Points (1pt = 1/72 of 1in)

Practical Examples

Typography

/* Responsive font sizes */ h1 { font-size: clamp(2rem, 5vw, 4rem); } p { font-size: max(1rem, 2vw); line-height: 1.6em; }

Layout & Spacing

/* Responsive container */ .container { width: min(1200px, 90vw); margin: 0 auto; padding: 2rem clamp(1rem, 5%, 3rem); } .grid { gap: min(2rem, 4vw); }

Responsive Design

/* Fluid hero section */ .hero { height: min(600px, 100vh); padding: min(5rem, 10vh) 1rem; } .card { width: min(400px, 100% - 2rem); }
